# Food Web

## 한 줄 소개

---

사용자들이 맛집들을 추천해주는 웹 서비스

## 📖 서비스 내용

- 사용자들이 먹고 맛있다는 생각하는 음식점을 추천해주는 사이트입니다.

## ⭐  핵심 기능

- 무한 스크롤링
- NextJs를 이용한 SSR 활용
- Kakao 지도 API, Daum 주소 API를 활용한 지도에 음식점 찾거나 지정.

## 🛠 기술 스택

### Front-End

- ReactJs, TypeScript
- NextJs
- Styled Component
- Redux-toolkit
- Redux(Saga)
- Eslint, Prettier
- Bable, Webpack

### Back-End

- NodeJs
- Sequelize
- Express

### DB

- MySQL
